Data Flow

How a filter change reaches the screen

1
User changes a filter FilterBar.vue updates shared ref in useFilters() composable (singleton — same instance across all views)
2
View detects the change Each view watch()es getCurrentFilters and calls loadData()
3
api.js builds the request Filter values serialised into URL query params (URLSearchParams). "all" values are omitted.
4
FastAPI filters in memory apply_filters() and filter_by_month() helpers applied to the startup-loaded JSON data
5
Pydantic serialises the response Filtered list returned as JSON; Vue reactive refs update; computed properties re-derive; template re-renders

Technical Debt — inherited from previous vendor

Known issues at handoff (status as of April 2026)

RESOLVED
Reports module defects (8 issues)

Filters not wired, Options API inconsistency, hardcoded strings, console noise, direct axios calls, fulfillment rate calculation error. All resolved in this engagement.

NOTED
No database — flat JSON files only

Data is loaded into memory at server startup. Changes do not persist across restarts. Suitable for current scale; will require a database migration before multi-user writes or data volume exceeds ~50K records.

NOTED
No authentication or authorisation

The application has a mock auth composable (useAuth.js) but no real session management or API protection. All endpoints are publicly accessible. Not suitable for internet-facing deployment without adding an auth layer.

NOTED
CORS allows all origins

Backend is configured with allow_origins=["*"]. Must be restricted to specific origins before any production deployment.

NOTED
No CI/CD pipeline

No automated build, test, or deploy process exists. The Playwright test suite added in this engagement (R3) can be integrated into a CI pipeline — recommended before any future vendor onboarding.

NOTED
Spending endpoints have no filter support

/api/spending/* endpoints do not accept warehouse or category filters. The Finance view shows global data only. Out of scope for this engagement but worth tracking.
