Known Gaps & Risks

Backlog route unregistered

Backlog.vue exists and works but is not in the Vue Router config in main.js. It's inaccessible from normal navigation.

Spending & Demand unfiltered

Spending and Demand views ignore the global filter bar. Users expect consistent filter behaviour across all views.

No database — no persistence

All data resets to JSON files on server restart. Tasks and purchase orders created at runtime are lost on restart.

No authentication

useAuth always returns isAuthenticated: true. All API endpoints are fully open. Not suitable for production without auth layer.

Hardcoded quarters (2025 only)

Quarterly report logic in main.py uses string matching against 2025 dates. No support for other years without code change.

Options API remnants

Reports.vue was using the old Options API at handoff. Now migrated. No other views affected — migration otherwise complete.
