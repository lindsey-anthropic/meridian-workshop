## Halcyon Logistics — Warehouse Operations Dashboard Remediation
*Vue 3 / FastAPI / 2025*

Halcyon engaged us to audit and remediate a Vue 3 dashboard delivered by a previous vendor that had left the project mid-stream. Scope included filter remediation, API pattern standardization, and introduction of Playwright end-to-end tests where none existed. We delivered a complete defect audit in Week 1, resolved 14 issues over three sprints, and handed off a test suite covering all primary user flows. Halcyon IT approved the first post-engagement release within two weeks of handoff — the first approved change in eight months.

**Relevance:** Directly mirrors R1 and R3. Same stack, same remediation-first engagement structure.

## Vanta Industrial — Inventory Reorder Automation
*React / Node / 2024*

Vanta's operations team needed a purchase recommendation engine integrated into their existing inventory system. We designed the recommendation algorithm with their procurement team (budget ceiling, reorder thresholds, demand signals), built the backend service, and delivered a frontend view that their team adopted as a primary daily tool within the first month. The engagement ran 8 weeks from kickoff to production.

**Relevance:** Directly mirrors R2. Different frontend stack, same product problem — translating inventory data and demand signals into actionable purchase recommendations.

---

## Nakamura Parts — Multi-locale Dashboard Internationalization
*Vue 3 / 2024*

Nakamura's Tokyo and Osaka teams were working in English-only views of a dashboard originally built for a US market. We extended an existing vue-i18n implementation to cover five previously untranslated modules, produced Japanese translations in collaboration with Nakamura staff, and delivered the work in a single 3-week sprint without disrupting the main development branch.

**Relevance:** Directly mirrors D2. Same technology (Vue 3 + vue-i18n), same operational context (non-English warehouse staff working in English-only tools).
