Requirement R2

Restocking
Recommendations

How it works

Operator enters a budget ceiling. The view calculates ranked purchase order recommendations per warehouse based on stock levels and demand forecast.

Data sources

Current stock from /api/inventory. Demand forecast from existing /api/demand endpoint. No new data pipeline required.

What it isn't

Not a procurement workflow. Recommendations are advisory — operators review and act. No approval integrations in scope.

Outcome

Production-ready Restocking view integrated into dashboard navigation, consistent with existing UI patterns and filter system.
