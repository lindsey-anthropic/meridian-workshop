# Pricing

**RFP #MC-2026-0417 — Meridian Components Inventory Dashboard Modernization**

---

## Fixed-Fee Proposal

We propose a **fixed fee of $72,500** for the full engagement as scoped in our Technical Approach, covering R1–R4 and D2 within a 10-week delivery schedule.

This figure is based on our review of the actual codebase and handoff documentation. It is not a top-down estimate — it reflects what we found when we looked at the code.

---

## What's Included

| Item | Included |
|------|----------|
| R1 — Reports remediation (up to 12 defects) | ✓ |
| R2 — Restocking recommendations view | ✓ |
| R3 — Playwright browser test suite (Reports + Restocking) | ✓ |
| R4 — Architecture documentation | ✓ |
| D2 — Internationalization (remaining modules, Japanese) | ✓ |
| D3 — Dark mode (stretch, best effort in Week 9) | ✓ if timeline permits |

---

## What's Not Included

| Item | Status |
|------|--------|
| D1 — UI modernization / visual redesign | Optional add-on (see below) |
| Database migration or infrastructure changes | Out of scope |
| R1 defects beyond 12 (if audit finds more) | Scope conversation at Week 1 |
| Ongoing support / hosting post-handoff | Available as separate retainer |

---

## Optional Add-On: UI Modernization (D1)

A full visual refresh would require a design phase and additional frontend build time. We estimate **$12,000–$18,000** as a fixed add-on, dependent on the scope of changes Meridian wants (component-level refresh vs. full design system). We recommend deferring this to a follow-on phase after core delivery.

---

## Assumptions

- Engagement begins approximately **June 2, 2026**, following contract execution in May.
- Meridian provides reasonable access to the operations team (Week 4) and IT team (Week 1, Week 10) as described in the Timeline.
- The application remains on its current stack (Vue 3 / FastAPI). No infrastructure provisioning is in scope.
- Fixed fee covers all labor. Expenses (travel, if any) billed at cost with prior approval.

---

## Why Fixed Fee

Given the budget predictability Meridian cited as a priority, we structure this as a fixed fee rather than T&M. You know the number upfront; we own the risk if our estimate was wrong. The one exception is if the Week 1 audit reveals the Reports defect count is materially larger than twelve — in that case we will surface the finding immediately and agree a revised scope before proceeding, rather than quietly absorbing cost or quietly cutting corners.
