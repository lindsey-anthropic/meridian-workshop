# Pricing

**RFP #MC-2026-0417 — Meridian Components Inventory Dashboard Modernization**

---

## Option A — Required Items Only (R1–R4)

**Fixed fee: $52,000**

Covers all four required items delivered over 6 weeks per the Phase 1 timeline. Fixed fee means no surprises — if the R1 audit surfaces more defects than the RFP indicates, we absorb the additional effort within this price.

| Item | Scope | Est. Effort |
|---|---|---|
| R1 — Reports remediation | Audit + full defect resolution | 3 weeks (inc. audit) |
| R4 — Architecture documentation | Codebase review + HTML deliverable | Concurrent with R1 audit |
| R2 — Restocking feature | Backend endpoint + Vue frontend view | 2 weeks |
| R3 — Automated browser tests | Playwright suite for R1 + R2 flows | 1 week |
| **Total** | | **6 weeks** |

## Option B — Required + Desired (R1–R4 + D1–D3)

**Fixed fee (Phase 1): $52,000**
**T&M not-to-exceed (Phase 2): $26,000**
**Combined not-to-exceed: $78,000**

Phase 2 is billed time-and-materials at $185/hour, not to exceed $26,000. If D1–D3 take less time than estimated, Meridian pays less. If scope creep appears, we flag it before exceeding the ceiling — we do not invoice above the NTE without a written change order.

## What's Included

- All development, testing, and documentation work described in the Technical Approach
- Two stakeholder checkpoints (Week 1 defect inventory; Week 3 Restocking mockup review)
- Architecture documentation deliverable (R4) formatted for Meridian IT
- Playwright test suite (R3) with plain-language descriptions suitable for IT review
- One round of revisions per deliverable based on Meridian feedback

## What's Not Included

- Deployment infrastructure or hosting setup
- ERP, WMS, or third-party system integrations
- Ongoing support or maintenance post-delivery (available under a separate retainer)
- Additional locales beyond Japanese (D2) — each additional locale is $2,500

---

## Payment Terms

- 30% on contract signing
- 40% on Phase 1 delivery and acceptance
- 30% on final delivery (Phase 1 only) or Phase 2 delivery and acceptance (Option B)

Fixed-fee invoices are milestone-based. T&M invoices are monthly, net 30.
