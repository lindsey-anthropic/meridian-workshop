# Pricing

**RFP MC-2026-0417 — Meridian Components Inventory Dashboard Modernization**

We propose fixed-fee pricing for all deliverables. No budget ceiling was provided by Meridian; our pricing reflects the scope defined in the technical approach and timeline sections. Rates are based on a blended team rate of approximately $175/hour. Change requests outside the defined scope will be estimated and approved in writing before work begins.

---

## Required Deliverables (R1–R4)

| Item | Description | Fixed Fee |
|------|-------------|-----------|
| R4 | Architecture review and documentation | $5,500 |
| R1 | Reports module audit and full remediation | $12,000 |
| R2 | Restocking view — backend + frontend | $21,000 |
| R3 | Automated browser test suite (Playwright) | $8,000 |
| — | Project management, handoff package, final delivery | $2,500 |
| **Total** | **Required scope (8 weeks)** | **$49,000** |

---

## Payment Schedule

| Milestone | Amount |
|-----------|--------|
| Contract execution | 30% ($14,700) |
| Phase 1 complete — architecture docs + defect baseline delivered | 25% ($12,250) |
| Phase 2 complete — Reports remediation verified | 25% ($12,250) |
| Phase 3 complete — all R1–R4 deliverables handed off | 20% ($9,800) |

Optional enhancements (D1–D4) billed 50% at start of Phase 4, 50% at completion.

---

## Assumptions

- Pricing is based on a blended rate of approximately $175/hour, consistent with mid-market US software consultancy rates
- Assumes access to the existing codebase and a functioning local development environment from day one
- Travel is not included; all work is delivered remotely
- Japanese translations for D2 are out of scope — provided by Meridian or a third-party translation service
- Scope changes or newly discovered requirements beyond what is documented in this proposal will be assessed and priced via written change request before work begins
