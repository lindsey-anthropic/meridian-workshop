# Pricing
**RFP #MC-2026-0417 — Inventory Dashboard Modernization**

---

We propose a phased fixed-fee structure. Each phase is priced and contracted independently. Meridian is not obligated to contract optional phases, and the required phases deliver a complete, functional system on their own.

---

## Required Phases

| Phase | Scope | Fixed Fee |
|-------|-------|-----------|
| Phase 1 | R4 Architecture Documentation + R1 Reports Remediation | $18,000 |
| Phase 2 | R2 Restocking Recommendations + R3 Automated Browser Testing | $32,000 |
| **Required Total** | | **$50,000** |

---

## Optional Phases

| Phase | Scope | Fixed Fee |
|-------|-------|-----------|
| Phase 3 | D1 UI Modernization | $14,000 |
| Phase 4 | D2 Internationalization | $8,000 |
| Phase 5 | D3 Dark Mode | $6,000 |
| **Optional Total** | | **$28,000** |

---

## Combined Not-to-Exceed

| Scenario | Total |
|----------|-------|
| Required phases only (P1 + P2) | $50,000 |
| Required + all optional (P1–P5) | $78,000 |

---

## What's Included

All phases include:
- All development and testing work
- A review call with Meridian stakeholders at phase close
- Handoff documentation for any new feature or significant change

Phase 3 (D1) pricing assumes Meridian provides brand guidelines by the start of that phase. If a brand-application pass is required after initial delivery due to delayed guidelines, it will be scoped and priced as a separate addendum.

---

## Payment Terms

- 30% due at contract execution
- 40% due at Phase 1 acceptance
- 30% due at Phase 2 acceptance
- Optional phases billed at 50% on phase start / 50% on acceptance

---

## Assumptions

- Scope is as defined in the Technical Approach and Timeline documents
- Changes to agreed scope are handled via written change order prior to work beginning
- Meridian's existing infrastructure (servers, hosting, deployment) is outside scope
- No third-party licensing costs are anticipated; all tools and frameworks used are open source
