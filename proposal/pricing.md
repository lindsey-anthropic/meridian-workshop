## Assumptions

The fixed fees above are contingent on the following assumptions. Items outside this scope are subject to a formal change-order process before work begins.

1. The JSON data layer is retained; no database design or migration is in scope
2. No infrastructure provisioning, hosting, or deployment is in scope
3. Meridian provides feedback at each milestone within 3 business days
4. UI direction for D1 is either client-supplied or approved from a vendor-proposed direction prior to implementation
5. Defects discovered outside the Reports module during the engagement are assessed separately
